django-admin startproject NOMBRE_PROJECTO           Crea un nuevo proyecto usando la libreria de django
py manage.py startapp NOMBRE_APP                    Crea una nueva aplicacion usando el proyecto
py manage.py runserver                              Inicializa el servidor de nuestro proyecto
py manage.py showmigrations                         Muestra todas las migraciones que tengamos en el proyecto
py manage.py sqlmigrate NOMBRE_APP NUM_MIGRA        Muestra el SQL de la migracion
py manage.py migrate                                Ejecuta todas las migraciones faltantes en la base de datos
py manage.py createsuperuser                        Crea un superusuario para que pueda acceder al panel administrativo
py manage.py makemigrations NOMBRE_APP --name NOMBRE_MIGRATION  Crea una nueva migracion sobre los cambios realizados
